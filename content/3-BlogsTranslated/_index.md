###  [Blog 1 - Optimizing Generative AI Video Inference](3.1-Blog1/)
This blog summarizes how Synthesia optimized its AI workflow by transitioning from a sequential video decoding process to an asynchronous architecture on Amazon EC2 G7e instances, helping eliminate GPU bottlenecks and significantly reduce costs.

###  [Blog 2 - Modernizing ML Pipelines for Agricultural Robots with Aigen](3.2-Blog2/)
This blog explores how Aigen transitioned from on-premise infrastructure to an AWS cloud-native architecture using Amazon SageMaker AI. This shift automated data labeling, 4-tier model training, and scaled computational capabilities for autonomous agricultural robots.

###  [Blog 3 - GPU Cost Attribution in Amazon EKS](3.3-Blog3/)
This blog introduces a solution to clearly identify and attribute GPU costs for individual teams sharing an Amazon EKS cluster. By combining OpenTelemetry, Prometheus, and Grafana, enterprises can solve the problem of wasted resources and realize FinOps practices in the AI/ML era.
